RockPaper
=========

RockPaper - 一个iOS上的剪刀石头布游戏。

# 介绍

一个iOS上的剪刀石头布游戏。在一方玩家选定手势后，另一方只能看到对方已经选择完毕的提示，只有在双方都选定手势后，系统才给出胜负结果。

[__进入应用商店__](http://itunes.apple.com/cn/app/rockpaper/id546275630?ls=1&mt=8)

# 操作说明

通过蓝牙，同身边的iOS移动设备连接后便可进行游戏。
